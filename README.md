== Installation ==

= Automatic Installation =

1. Go to **Plugins > Add New** in your WordPress admin
2. Search for "CopySpell AI"
3. Click **Install Now** and then **Activate**
4. Navigate to **CopySpell AI** in your admin menu to configure

= Manual Installation =

1. Download the plugin ZIP file
2. Go to **Plugins > Add New > Upload Plugin**
3. Select the ZIP file and click **Install Now**
4. Activate the plugin
5. Go to **CopySpell AI** settings to configure your API keys

= Configuration =

1. **Get Your API Keys**
   * Google Gemini: Visit [Google AI Studio](https://aistudio.google.com/) to get a free API key
   * Groq: Visit [Groq Console](https://console.groq.com/) for a free API key

2. **Enter API Keys**
   * Go to CopySpell AI > Settings
   * Add your API keys in the respective fields
   * Click "Validate" to verify each key

3. **Set Your Preferences**
   * Configure your default brand tone
   * Set preferred content length
   * Choose your target audience defaults

4. **Start Generating**
   * Go to any WooCommerce product edit page and click "CopySpell AI"
   * Or use Bulk Generation to process multiple products at once

CopySpell AI is designed with your privacy in mind. Here's exactly how your data is handled:

**How It Works:**
When you generate content, your product data is sent to our proxy server at copyspell.actusanima.com. Our server then calls the AI providers using your API keys and returns the response to you.
